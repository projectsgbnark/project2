# GEE BEE NETWORK PVT LTD Website

This project is a simple Node.js application that serves a static website for GEE BEE NETWORK PVT LTD.

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   npm start
   ```
3. Open in browser:
   ```
   http://localhost:3001
   ```

## Features

- Static website served from `index.html`
- Responsive design via `styles.css`
- Contact form handled by Express at `/api/contact`
